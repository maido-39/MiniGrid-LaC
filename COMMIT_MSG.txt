dev-memory: VLM-only script, lazy minigrid, memory dict formatting

- run_memory_dev.py: default -p dev_prompt.txt, -i dummy_img.png; user_prompt
  optional, read from terminal when omitted; _render_system_prompt override for
  dev-memory local prompt files (no render_prompt_from_string in lib).
- utils/__init__.py: lazy-load map_manager (MiniGridEmojiWrapper,
  load_emoji_map_from_json) via __getattr__ so dev-memory does not load
  gymnasium/minigrid.
- memory_renderer.py: dict values render with leading newline and per-line
  indent so "- Task Process: $memory[task_process]" shows goal/status/blocked_reason
  on separate indented lines.
- global_variables.py: USE_VERBALIZED_ENTROPY = False.
- prompt_interp.py: remove trailing blank line.
